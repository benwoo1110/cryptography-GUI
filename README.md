# cryptography-GUI
A Simple visualisation for basic string cryptography algorithms.

List of algorithms:
* Caesar Cipher
* Monoalphabetic Cipher
* Homophonic Substitution Cipher
* Polygram Substitution Cipher
* Polyalphabetic Substitution Cipher

Depencencies needed:
* python 3
* pygame

pygame installation:
```pip install pygame```
