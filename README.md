A Simple visualisation for basic string cryptography algorithms.

## List of algorithms:
* Atbash Cipher
* Affine Cipher
* Caesar Cipher
* Monoalphabetic Cipher
* Polyalphabetic Substitution Cipher

## Depencencies needed:
* python 3.x
* pygame
* yyaml

#### Manual install: 
```css
pip3 install pyyaml pygame
```

## To run to the program:
#### For macos/linux:
run `start.sh` file
```css
chmod +x start.sh
./start.sh
```
#### For windows:
run `start.bat` file
```css
.\start.bat
```
