# cryptography-GUI
A Simple visualisation for basic string cryptography algorithms.

List of algorithms:
* Atbash Cipher
* Affine Cipher
* Caesar Cipher
* Monoalphabetic Cipher
* Polyalphabetic Substitution Cipher
* Rail-fence Cipher

Depencencies needed:
* python 3.x
* pygame

pygame installation: `pip install pygame`
